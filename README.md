# ClaimAgent – AI Motor Insurance Claim Estimator

ClaimAgent is an AI-powered prototype that analyzes car damage images and generates instant repair cost estimates for motor insurance claims.

The system detects damaged parts using computer vision techniques and generates a structured claim report with cost breakdowns.

## Features

• Upload car damage images  
• Computer Vision damage detection  
• AI confidence scoring  
• Estimated repair cost calculation  
• Automated claim report generation  
• PDF report download  
• Multi-angle vehicle damage analysis  

## Tech Stack

Frontend
React + Vite  
TailwindCSS  

Backend
Node.js  
Express  
TypeScript  

Database
MongoDB  

Libraries
OpenCV concepts, React Query, Radix UI

## Workflow

1. User enters vehicle details  
2. Uploads multiple car damage images  
3. AI analyzes images and identifies damaged parts  
4. System estimates repair cost  
5. Final claim report is generated
